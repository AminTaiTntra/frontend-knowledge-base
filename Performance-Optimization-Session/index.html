What is Preload?

Preload tells the browser to fetch critical resources immediately, ensuring important assets load as early as possible in the page lifecycle.

<link rel="preload" href="critical.css" as="style"> <link rel="preload" href="hero.jpg" as="image">

Perfect for:

  • Above-the-fold images
  • Critical CSS files
  • Web fonts
  • Hero banners
  • 40% faster initial load

What is Prefetch?

Prefetch downloads resources that users will likely need next, during browser idle time, making future interactions instant.

<link rel="prefetch" href="/products.html"> // Smart hover prefetch link.addEventListener('mouseenter', () => { prefetchPage(link.href); });

Perfect for:

  • Next page in workflow
  • Popular product pages
  • User dashboard
  • Search results
  • 70% faster navigation
  • Hover-based prefetching
  • Smart predictions
  • Network optimization

What is Intersection Observer?

A modern API that efficiently detects when elements enter the viewport, replacing expensive scroll event listeners.

// Create observer const observer = new IntersectionObserver((entries) => { entries.forEach(entry => { if (entry.isIntersecting) { loadImage(entry.target); } }); });

Perfect for:

  • Lazy loading images
  • Infinite scroll
  • Animation triggers
  • Analytics tracking
  • 90% less CPU usage
  • Lazy image loading
  • Scroll animations
  • Performance comparison
